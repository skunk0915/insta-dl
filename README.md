# Insta-DL 利用方法マニュアル

Insta-DL をご利用いただきありがとうございます。このツールは、Instagram の動画（リールなど）を最高画質で簡単にダウンロードするためのウェブアプリケーションです。

---

## 1. 起動方法

### 初回・通常起動
フォルダ内にある `run.bat` をダブルクリックしてください。
黒い画面（コマンドプロンプト）が立ち上がり、以下のメッセージが表示されれば準備完了です。
`INFO: Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)`

※この画面が開いている間だけ、ツールを利用できます。終了したい場合は、画面を閉じるか `Ctrl + C` を押してください。

---

## 2. 操作手順

1.  **ブラウザで開く**
    ブラウザ（Chrome, Edgeなど）の検索バーに以下のURLを入力して開きます。
    [http://localhost:8000](http://localhost:8000)

2.  **URLを貼り付ける**
    Instagramで保存したい動画の「リンク」をコピーし、入力欄に貼り付けます。
    例: `https://www.instagram.com/reel/XXXXXXXXXXX/`

3.  **解析を開始する**
    「動画をダウンロード」ボタンをクリックします。
    ※解析には数秒〜15秒ほどかかる場合があります。

4.  **保存する**
    解析が終わると、動画のサムネイルと「準備完了！保存する」という緑色のボタンが表示されます。
    ボタンをクリックすると、動画ファイル（MP4形式）がダウンロードされます。

---

## 3. トラブルシューティング

### 画面が開かない場合
- `run.bat` が正しく実行されているか確認してください。
- 他のアプリがポート `8000` を使用している可能性があります。

### ダウンロードに失敗する場合
- **URLを確認**: 非公開アカウントの投稿はダウンロードできません。公開されている投稿のURLを使用してください。
- **仕様変更への対応**: Instagramのアップデートにより取得できなくなることがあります。その場合は、コマンドプロンプトやターミナルで以下のコマンドを入力し、エンジンを最新版に更新してください。
  ```powershell
  pip install -U yt-dlp
  ```

---

## 4. 仕様と制限
- **保存先**: ブラウザの標準設定（通常は「ダウンロード」フォルダ）に保存されます。
- **ファイル削除**: サーバー（downloadsフォルダ）に一時保存された動画データは、1時間後に自動で削除されます。
- **ファイル形式**: 基本的に高品質な `.mp4` 形式で保存されます。
